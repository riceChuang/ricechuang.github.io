Day 2 · 4/24（四）
換錢 → 廣藏市場 → 景福宮 → 南山塔
行程最豐富，建議9:00出門
💰 換錢
09:00–10:00
推薦順序：Money Box（明洞最優匯率）→ 大信換錢所 → 大使館前換錢所。建議先換大部分，避免後面跑回來。
📍 Naver 搜尋「명동 환전소」
前往廣藏市場
飯店步行18分鐘，或搭計程車10分鐘（約3,000–4,000韓元）。
地鐵：忠武路 →（4號線）→ 東大門歷史文化公園 →（2號線）→ 乙支路4街站
🥟 廣藏市場
10:00–12:00
必吃：156號攤（綠豆煎餅）、214號攤（馬格利米腸）。帶現金，市場內刷卡不方便。
📍 Naver Map 廣藏市場
前往景福宮
乙支路4街 →（2號線）→ 市廳站 → 換5號線 → 光化門站，步行5分鐘抵景福宮正門。
🏯 景福宮
12:30–14:30
入場費3,000韓元（週二休）。入口附近可租韓服免費入場！必看：勤政殿、香遠亭、慶會樓。
📍 Naver Map 景福宮
☕ Cafe Onion 安國店
14:30–15:30
景福宮出來步行10分鐘。必點：芝士貝果、米麻糬麵包。建議避開下午尖峰。
📍 Naver Map Cafe Onion
🛍️ 明洞逛街
15:30–17:00

📍 Naver Map 明洞商圈
🗼 南山塔（N首爾塔）
17:00–18:30
從明洞搭纜車上山（單程9,500韓元），黃昏最美。可掛同心鎖。回程搭05番公車下山。
📍 Naver Map 南山纜車站
🏬 樂天百貨明洞店
18:30–20:00
從南山下來步行可達。地下美食街晚餐、免稅店、超市採購伴手禮。
📍 Naver Map 樂天百貨
這天行程最密！景福宮和南山塔如果體力不夠可以二擇一。換錢優先在 Money Box，匯率最優。

Day 3 · 4/25（五）
弘大 → 延南洞
文青咖啡廳慢遊
交通：飯店 → 弘大
忠武路 →（4號線）→ 首爾站 → 換機場鐵路AREX → 弘大入口站，約25–30分鐘。
或計程車約20分鐘，費用12,000–15,000韓元。
🍳 Egg Drop 弘大店
09:30–10:30
人氣蛋捲三明治，必點：牛肉蛋捲、BLT蛋捲。建議開門就到，避開排隊。
📍 Naver Map Egg Drop
🍞 鹽麵包（소금빵）
10:30–11:00
弘大附近多家名店，推薦 Pain de Papa 或 Cafe Onion 弘大店，外酥內軟奶油爆漿。

🎨 弘大街頭 / 獨立小店
11:00–13:00
弘大塗鴉街、獨立品牌店、唱片行。週末有街頭表演，氣氛很好。
📍 Naver Map 弘大
步行約15分鐘前往延南洞
🌿 延南洞散步
13:00–17:00

延南洞很適合邊逛邊找吃的！午餐就地解決，這區韓式、西式選擇都多。

Day 4 · 4/26（六）
馬場洞韓牛 → 聖水洞 → 東大門
美食 + 潮流的一天
交通：飯店 → 馬場洞
忠武路 →（4號線）→ 東大門歷史文化公園 →（5號線）→ 馬場站，約15分鐘。
🥩 馬場洞韓牛市場
10:00–13:00
首爾最大肉市場！在1樓市場選肉，拿到2樓烤肉店加工（加工費約3,000–5,000韓元/人）。必吃：嫩肩里肌（꽃등심）。帶現金！
📍 Naver Map 馬場洞肉類市場
前往聖水洞
馬場站 →（5號線）→ 王十里站 → 換2號線 → 聖水站，約10分鐘。
🏭 聖水洞（성수동）
13:30–16:30
首爾最潮工業風街區！必去：大林倉庫（대림창고）、各種快閃品牌店、塗鴉牆。週末人多，提早到。
📍 Naver Map 聖水洞
前往東大門
聖水站 →（2號線）→ 東大門歷史文化公園站，約8分鐘。
🌃 東大門夜逛 + DDP
17:00–21:00
東大門設計廣場（DDP）夜景必拍！Doota Mall、周邊批發市場天黑後更熱鬧。
📍 Naver Map 東大門DDP
馬場洞一定要帶現金！聖水洞建議留3小時，很多咖啡廳藏在巷子裡需要時間找。

Day 5 · 4/27（日）
江南 → COEX → 狎鷗亭 → 返台
⚠️ 19:00 飯店集合，務必控制時間！
⚠️ 最後一天！建議Check-out後請飯店代保管行李，18:00前一定要從狎鷗亭出發回飯店。

交通：飯店 → COEX
忠武路 →（4號線）→ 東大門歷史文化公園 → 換2號線 → 三成站，約30分鐘。
📚 COEX Mall · 星空圖書館
10:00–13:00
別마당도서관（星空圖書館）免費參觀，網美必打卡！建議平日早上去，週末人超多。同場逛COEX購物、吃午餐。
📍 Naver Map 星空圖書館
前往狎鷗亭
三成站 →（2號線）→ 狎鷗亭站，約8分鐘。
💄 狎鷗亭羅德奧街
13:00–17:00
韓星出沒地帶！精品店、獨立咖啡廳聚集。必逛：清潭洞街、島山公園周邊咖啡廳。最後採購伴手禮。
📍 Naver Map 狎鷗亭羅德奧
🚐 飯店集合搭車
⚠️ 19:00 務必到
從狎鷗亭回飯店：地鐵約35分鐘，計程車約25分鐘（15,000–20,000韓元）。建議18:00出發，留時間整理行李。
狎鷗亭消費較高，記得保留足夠韓元。OLIVE YOUNG D2就買完了不用再跑！
